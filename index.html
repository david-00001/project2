Project Description

In 2020 the US along with the rest of the world experienced a tremendous upset in lifestyle that reached virtually every person. Daily news updates communicated the spread of COVID as well as the responses from state governments. Several websites including the CDC produced real-time information to help people understand the spread of this new pandemic.

This dashboard was created in fulfillment of a class project to create a dashboard that makes use of JavaScript, Flask, D3, and other utilities learned.

Our objective was to create a dashboard that allowed a user to explore the spread of COVID in the US both geographically and by race/ethnicity.
